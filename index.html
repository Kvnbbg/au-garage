Installation Guide for Developers

1

Prerequisites

Ensure you have the following installed on your system:

  • Node.js (v16 or higher recommended)
  • npm (comes with Node.js)
  • PostgreSQL (or your preferred database system)
  • Git (for cloning the repository)
2

Clone the Repository

Clone the au-garage repository from GitHub:

git clone https://github.com/Kvnbbg/au-garage

Then navigate to the project directory:

cd au-garage
3

Install Dependencies

Install all required dependencies:

npm install
4

Configure Environment

Create a .env file in the root directory with the following variables:

DB_HOST=your_database_host
DB_USER=your_database_user
DB_PASSWORD=your_database_password
DB_NAME=your_database_name
PORT=your_preferred_port
JWT_SECRET=your_jwt_secret_key
SMTP_HOST=your_smtp_host
SMTP_PORT=your_smtp_port
SMTP_USER=your_smtp_user
SMTP_PASS=your_smtp_password
SMS_API_KEY=your_sms_api_key
5

Initialize Database

Run database migrations to set up the schema:

npm run migrate
6

Start the Application

For production:

npm start

For development (with hot reload):

npm run dev
7

Access the Application

Open your browser and navigate to:

http://localhost:[PORT]

(Replace [PORT] with the port number you configured in your .env file)

Getting Started for Users

1

Registration & Login

After installation, access the application and:

  • Register an admin account (first user is automatically granted admin privileges)
  • Subsequent users can register or be added by admin
  • Log in with your credentials to access the dashboard
2

User Roles & Permissions

The system supports three role types:

  • Admin: Full access to all features and user management
  • Staff: Access to customer management, appointments, and inventory
  • Mechanic: Access to assigned work orders and vehicle service history

Key Features

Vehicle Tracking

Track VIN, mileage, and complete service history for all vehicles in your garage.

Appointment Scheduling

Schedule appointments, assign mechanics, and set automated reminders.

Inventory Management

Manage parts inventory with stock alerts and reordering capabilities.

Invoice Generation

Generate PDF invoices with integrated payment processing.

Customer Communication

Automated email/SMS notifications for appointments and service updates.

Analytical Reporting

Comprehensive reports on business performance and garage operations.
