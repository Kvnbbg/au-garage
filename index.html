Installation Guide for Developers

1

Prerequisites

Ensure you have the following installed:

  • Python 3.8 or higher
  • pip (Python package manager)
  • PostgreSQL (database system)
  • Git (for cloning the repository)
2

Clone the Repository

Clone the au-garage repository from GitHub:

git clone https://github.com/Kvnbbg/au-garage

Navigate to the project directory:

cd au-garage
3

Set Up Virtual Environment

Create and activate a virtual environment:

python3 -m venv venv
source venv/bin/activate # Unix/macOS
# For Windows: venv\Scripts\activate
4

Install Dependencies

Install dependencies from requirements.txt:

pip install -r requirements.txt
5

Configure Environment

Create a .env file in the root directory:

FLASK_APP=app.py
FLASK_ENV=development
DATABASE_URL=postgresql://user:password@localhost:5432/garage_db
SECRET_KEY=your_secret_key
SMTP_HOST=your_smtp_host
SMTP_PORT=your_smtp_port
SMTP_USER=your_smtp_user
SMTP_PASS=your_smtp_password
SMS_API_KEY=your_sms_api_key

*Ensure PostgreSQL is running and the database `garage_db` is created.*

6

Initialize Database

Run migrations to set up the schema:

flask db upgrade
7

Start the Application

For development:

flask run

For production:

gunicorn --workers 4 --bind 0.0.0.0:8000 app:app
8

Access the Application

Open your browser and navigate to:

http://localhost:5000

Or visit the production build at: https://web-production-d728.up.railway.app/

Getting Started for Users

1

Registration & Login

Access the application and:

  • Register an admin account (first user gets admin privileges)
  • Subsequent users can register or be added by admin
  • Log in to access the dashboard
2

User Roles & Permissions

The system supports three roles:

  • Admin: Full access to features and user management
  • Staff: Manage customers, appointments, and inventory
  • Mechanic: View assigned work orders and service history

Key Features

Service Management

Track repairs, maintenance, and vehicle sales with detailed history.

Inventory Control

Real-time stock alerts and automated reordering for parts.

Customer Portal

Direct communication, appointment booking, and invoices.

Role-Based Access

Admin, staff, and mechanic permissions for secure access.

Analytics Dashboard

Performance metrics and reporting for business insights.
