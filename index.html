Installation Guide for Developers

  1. Prerequisites

    Ensure you have the following installed on your system:

    • Python 3.8 or higher
    • pip (Python package manager)
    • PostgreSQL (database system)
    • Git (for cloning the repository)

  2. Clone the Repository

    Clone the au-garage repository from GitHub:

    git clone https://github.com/Kvnbbg/au-garage

    Then navigate to the project directory:

    cd au-garage

  3. Set Up Virtual Environment

    Create and activate a virtual environment:

    python3 -m venv venv
    source venv/bin/activate # Unix/macOS
    # For Windows: venv\Scripts\activate

  4. Install Dependencies

    Install all required dependencies from requirements.txt:

    pip install -r requirements.txt

  5. Configure Environment

    Create a .env file in the root directory with the following variables:

    DATABASE_URL=postgresql://user:password@localhost:5432/garage_db
    SECRET_KEY=your_secret_key
    SMTP_HOST=your_smtp_host
    SMTP_PORT=your_smtp_port
    SMTP_USER=your_smtp_user
    SMTP_PASS=your_smtp_password
    SMS_API_KEY=your_sms_api_key

  6. Initialize Database

    Run database migrations to set up the schema:

    flask db upgrade

  7. Start the Application

    For development:

    flask run

    For production (using gunicorn):

    gunicorn -w 4 'app:app'

  8. Access the Application

    Open your browser and navigate to:

    http://localhost:5000

    Or visit the production build at: https://web-production-d728.up.railway.app/

Getting Started for Users

  1. Registration & Login

    After installation, access the application and:

    • Register an admin account (first user is automatically granted admin privileges)
    • Subsequent users can register or be added by admin
    • Log in with your credentials to access the dashboard

  2. User Roles & Permissions

    The system supports three role types:

    • Admin: Full access to all features and user management
    • Staff: Access to customer management, appointments, and inventory
    • Mechanic: Access to assigned work orders and vehicle service history
